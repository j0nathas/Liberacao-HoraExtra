# my-project

Projeto full-stack com React (frontend) e Node.js/Express (backend).

## Requisitos
- Node.js 20+
- Docker e Docker Compose (opcional)

## Como rodar

### Com Docker
```bash
docker-compose up
```

### Manualmente
```bash
# Backend
cd backend && npm install && npm run dev

# Frontend (outro terminal)
cd frontend && npm install && npm run dev
```

## Estrutura
- `frontend/` — React + Vite
- `backend/`  — Node.js + Express
